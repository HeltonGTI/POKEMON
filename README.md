# 🧢 App Pokémon

Aplicação web interativa que consome a **PokeAPI** para buscar informações de Pokémon, exibir os dados em uma tabela e armazenar os resultados no navegador.

---

## 📸 Preview

Interface simples, intuitiva e responsiva, permitindo ao usuário buscar e gerenciar Pokémon facilmente.

---

## 🚀 Funcionalidades

✔ Buscar Pokémon pelo nome
✔ Exibir dados diretamente da API
✔ Mostrar informações como:

* Nome
* Imagem
* Tipo
* Peso

✔ Armazenar dados no **LocalStorage**
✔ Listar Pokémon em tabela dinâmica
✔ Remover Pokémon da lista
✔ Confirmação antes de excluir um Pokémon

---

## 🛠️ Tecnologias Utilizadas
